Blogging in Malayalam Using Wordpress

Feb 14th, 2013 | Comments

To write blogs in malayalam, you can use the Google transliteration service.After that you can copy the content and paste it to the wordpress editor.

To read your malayalam blogs, you have to edit wp-config.php, which will be present in the root directory of the web hosting server, where you had installed the wordpress. Look for the following line :

define(‘DB_CHARSET’, ‘utf8′);

Comment the above line :

//define(‘DB_CHARSET’, ‘utf8′);

Save the file in the server and publish your blog.Now everyone can read your malayalam blogs.
